Summary of Stand-Alone Costs

  • Measurements confirm dominance of LSA processing cost

    • Processing (update/generation) of single LSA takes ? 0.9msec!

  • Computational cost is relatively small

    • Computation of QoS routing table takes ? 10msec for a 400 nodes network
    • Cost of on-demand computation varies with path length
      • About 1.2msec for a 4 hop paths
      • Pre-computation is cheaper when number of requests between pre-computations exceeds about 10
    • Costs of accessing topology database and selecting paths are minimal and fairly insensitive to network size
