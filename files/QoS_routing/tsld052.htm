Outline of Possible Randomized Path Selection

  • At path selection time

    • Compute a selection probability for each path depending on the available bandwidth on the path and its hop length

  • Probability of selecting a path should

    • Become smaller as the path gets longer
    • Become smaller as the path gets narrower
    • Depend on request size
      • Can afford to be more aggressive with small requests
