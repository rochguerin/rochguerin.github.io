Additional Features

  • Trunk Reservation

    • Used to control level of alternate routing
    • Long alternate paths are accepted based on load dependent threshold that grows with hop count

  • Load Balancing

    • Need to choose between equal length and bandwidth paths
    • Randomly pick the next path node, based on the amount of bandwidth available on the link to the next node
