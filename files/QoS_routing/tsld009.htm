Evaluation Environment

  • QoS routing for bandwidth guarantees only

    • Requested bandwidth needs to be available on all links of selected path

  • Link state protocol architecture

    • Resource updates are propagated using flooding

  • Explicit route model

    • Source node computes full path to destination
    • Path is “pinned” for the duration of the request
      • Forwarding state is installed
