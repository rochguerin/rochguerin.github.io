    QoS Extensions to OSPF

  • Based on GateD 4.0 and the “new” OSPF code
    • Extends topology database and LSA format to support TOS encoding of available bandwidth
  • Path computations models supported
    • Pre-computed ? (separate) QoS routing table
    • Computed on-demand
  • Change based update triggering (thresholds)
    • Parameterized by threshold value and hold-on timers
  • Simplifying assumptions
    • Single OSPF area
    • All routers run extended OSPF
