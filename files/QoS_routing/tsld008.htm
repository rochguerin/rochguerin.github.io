Cost of QoS Routing

  • Main departure from best-effort routing

    • More frequent and complicated path computation
    • More protocol traffic to maintain up-to-date resource availability information

  • Main questions

    • What factors determine this cost and how large is it?
    • How can we reduce cost and how does this affects routing performance?
    • How can we achieve good routing performance/cost tradeoffs?
