Evaluation Methodology

  • Develop QoS enabled version of OSPF

  • Benchmark key operations from implementation

    • Computation of QoS routing table
    • Processing of LSA
    • Access to topology database
    • Path selection (pre-computation)

  • Combine benchmarking and simulation to emulate operation in “large” network

    • Simulation trace logs events at router
    • Router load is “computed” based on trace and stand-alone costs
