    Some Minor Implementation Issues

  • OSPF implements hold down timers
    • We do not override them, just reduce their value
  • Different rules for processing incoming updates
    • An update does not necessarily mean a change in the topology
  • Acknowledgments of (QoS) LSAs are not necessary
    • Unreliable LSAs?
  • Route table memory de-allocation can be optimized
    • Reuse allocated memory and minimize deallocation
