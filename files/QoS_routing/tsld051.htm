Handling Large Hold-Down Timers

  • Large hold-down timers introduce arbitrary inaccuracies

    • No systematic model for inaccuracy

  • Possible approach

    • Consider bandwidth information only as a hint
    • Computing optimal paths is not necessarily meaningful
    • Avoid systematic selection of the best path
      • Will quickly become bad....
    • Solution: Introduce some randomization in the path computation/selection
