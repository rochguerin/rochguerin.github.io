Evaluation Scenarios

    We vary:

  • Network topology

    • ISP and mesh

  • Path computation

    • On-demand, pre-computed
    • Path pre-computation period

  • Traffic characteristics

    • Poisson arrival of requests and exponential holding times
    • Requests sizes (uniform in interval: 64kbps-5Mbps), uniform and non-uniform traffic

  • LSA generation methods

    • Triggering Policies
      • Threshold based
      • Class based
        • a hysteresis mechanism to avoid redundant updates
      • Existence and settings of hold-down timers
